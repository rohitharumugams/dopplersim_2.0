Speed (v₂)

Random render speed per clip.

Pass-by distance (generated clip)

CPA = closest point of approach — when the car passes nearest the microphone. h₂ is the side distance at that moment (meters). Each clip picks a random value in this range. Set min = max for a fixed distance on every clip.

Pass-by time (generated clip)

t_CPA₂ is the time (seconds from the start of the output clip) when the car is closest to the mic. Default 5 s (fixed). Set min = max to keep one value across all clips.

Source recording geometry

Fixed assumptions about your input WAV used to remove Doppler from the real pass-by (v₁ inversion). h₁ defaults to 0.5 m. t_CPA₁ is read per clip from the matching .txt sidecar (not set here).
